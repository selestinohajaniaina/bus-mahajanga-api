DESCRIPTION

Le bus-mj, abréviation pour "bus station à Mahajanga, Madagascar", représente une interface de programmation d'application (API) destinée à simplifier le processus de sélection du bon bus vers une destination spécifique pour les voyageurs étrangers. Cette API met à disposition une gamme d'informations utiles, telles que la liste exhaustive des bus disponibles, les différents arrêts desservis, les horaires de circulation, la durée totale du trajet, ainsi que d'autres détails pertinents. L'initiative de développer cette API a émergé lors de l'Hackathon Clean Code organisé dans le cadre du devfest2023 à Antananarivo, Madagascar. Ce projet vise à améliorer l'expérience des voyageurs en fournissant des données précises et accessibles pour faciliter leurs déplacements dans la région de Mahajanga.

Usage

Nous nous excusons pour le désagrément, car l'API n'est pas encore finalisée. Toutefois, nous vous encourageons vivement à suivre les 'ENDPOINTS' spécifiés afin d'obtenir les réponses nécessaires. Votre compréhension et votre coopération pendant cette phase de développement sont grandement appréciées.


ENDPOINTS:


  • https://bus-mj.onrender.com/api?depart=star&fin=poste : Retourne une objet qui indique les bus correspond, depart est l'arret du depart, et fin est celle d'arrive.

  • https://bus-mj.onrender.com/api/stop/all : Retourne une objet qui affiche tout les arrets du bus.

  • https://bus-mj.onrender.com/api/bus/all : Retourne une objet qui affiche tout les bus.

  • https://bus-mj.onrender.com/api/stop?id=3 : Retourne une objet qui affiche tout les arrets d'une bus specifie, id est l'identifiant du bus(BUS_ID).

Feature

Nous aspirons à enrichir davantage cette API en intégrant des fonctionnalités telles que l'affichage:

  • du trajet choisi sur Google Map
  • la localisation en temps réel du bus
  • la disponibilité de l'estimation du temps d'approche du bus
  • a possibilité d'installer l'API en tant que module avec des commandes standard telles que "npm i bus-mj" et "pip install bus-mj" pour une utilisation hors ligne

Nous lançons un appel aux designers et aux développeurs FrontEnd pour transformer notre API en une interface graphique conviviale. De même, nous sollicitons les développeurs BackEnd pour collaborer avec nous afin d'améliorer et d'optimiser les fonctionnalités de l'API. Votre expertise et votre contribution seront essentielles pour faire évoluer cette solution vers un outil plus complet et performant.
